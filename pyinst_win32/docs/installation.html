Installation¶

Linux¶

Follow these steps:

$ git clone http://www.github.com/tastyminerals/nn_search2.git
$ cd nn_search2
$ sudo linux_install.sh

Windows¶

Make sure you have Python2.7 installed and set as default Python interpreter in the system.

You can find nn_search2.exe installer in bins folder.
